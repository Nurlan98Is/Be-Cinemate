Cinemate API

Добро пожаловать в API от Cinemate

Здесь вы можете получить данные о сериалах или фильмах, главное правильно отправлять запросы на сервер!

Аутентификация

Залогиниться Login

https://be-cinemate.vercel.app/auth/login

Метод: POST

Зарегистрироваться

https://be-cinemate.vercel.app/auth/register

Метод: POST

Получение Users

Получить всех Юзеров

https://be-cinemate.vercel.app/users

Метод: GET

Получить Юзера По Емайлу или по Имени и Фамилии

https://be-cinemate.vercel.app/users/getUserByInfo

Метод: POST

Получить Юзера по ID

https://be-cinemate.vercel.app/users/getUserById

Метод: POST

Получить Юзера по NickName

https://be-cinemate.vercel.app/users/getUserByNickName

Метод: POST

Поулчение списков фильмов и сериалов

Получить список всех сериалов

https://be-cinemate.vercel.app/serials

Метод: GET
